# cucumber-jvm-rhino
Cucumber Rhino

## Installation

### Maven
To use cucumber-jvm-clojure in your project, add the following dependency to your `pom.xml`:
```xml
<dependency>
    <groupId>io.cucumber</groupId>
    <artifactId>cucumber-rhino</artifactId>
    <version>2.0.1</version>
    <scope>test</scope>
</dependency>
```
